# Monster PCs (Savage Species)

Rules for playing monsters as player characters. Source: *Savage Species* (2003).

---

## Core Concepts

### ECL Formula

> **ECL = Base HD + Level Adjustment (LA) + Class Levels**

- **Starting ECL** (at character creation) = HD + LA. Must equal the party's average character level.
- If Starting ECL < party average, add class levels until it matches.
- **Character Level** = HD + class levels (no LA). Governs BAB, saves, feats at every 3rd level.
- **ECL** governs XP costs, starting equipment, and XP needed to level.
- CR ≠ ECL. CR measures encounter threat for a few rounds; ECL measures party-resource equivalence across a whole adventure.

### Level Adjustment (LA)

A number added to HD to reflect special abilities (natural armor, SLAs, SAs, etc.) that make the creature more powerful than raw HD implies. A bugbear has 3 HD and LA +1, so a bugbear PC entering a 4th-level party adds 0 class levels; one entering a 5th-level party adds 1 class level.

**LA +0** = playable at 1st level like a standard race. **LA –** (negative) is treated as 0 in practice; a kobold (LA –1) is conventionally played with LA +0.

### Fractional / Replaced Hit Dice

If a monster has only 1 HD (or a fractional HD), the first class level taken *replaces* that racial HD. The character functions as a full 1st-level member of that class. This makes single-HD monsters (aasimar, tiefling, goblin, kobold) much easier to integrate.

### Feats

Monsters gain feats based on total HD (not class levels alone): 1 feat at 1st HD, then every 3rd HD thereafter (3rd, 6th, 9th…). Class levels count toward this total.

### Skills

Monster skill points use the monster type's points-per-HD formula (e.g., humanoids get (2 + Int mod) × HD). Class levels add the class's points per level thereafter.

### Monster Classes (Chapter 3)

Some large or powerful monsters (troll, ogre, mind flayer, etc.) can be played using a **monster class** — a structured progression that builds up to the full monster's abilities over multiple levels, starting CR 1 and ending at the monster's full CR. This lets a player start at 1st level in a campaign and grow into the monster. The monster class acts like a racial hit die track with class features unlocking gradually.

## Recommended Starting ECLs for Ptolus

Ptolus is an ECL 1–12 city campaign at its core. Practical guidance:

| Starting ECL | Good Monster Choices |
|---|---|
| 1 | Goblin, Kobold (treat LA as +0) |
| 2 | Aasimar, Tiefling, Mongrelfolk (from *Races of Destiny*) |
| 3 | Gnoll, Lizardfolk |
| 4 | Bugbear |
| 5–6 | Centaur, Ogre |
| 7–8 | Minotaur, Doppelganger |
| 9–10 | Gargoyle, Medusa |
| 11+ | Troll, Harpy, Yuan-ti — only if party is already high-level |

**Ptolus flavor note:** Doppelgangers are perfect for intrigue in a city of secrets. Tieflings fit the demonic history of the Spire. Lizardfolk (assari) already appear in Ptolus lore. Aasimars work well with the city's Good-aligned temples and the Keepers of the Veil.

---

## Practical DM Notes

- Require a **backstory** for why a monster race lives in/near Ptolus. The Brotherhood of Redemption vouches for some monsters; others are refugees or escaped dungeon denizens.
- **One monster PC per party** is a good soft rule. Two large creatures causes constant logistics problems (doors, taverns, city guards).
- Use **monster classes** for creatures above ECL 6 if you want smooth play from level 1. Raw Monster Manual + class levels works better for creatures joining mid-campaign.
- **Reducing LA:** Savage Species briefly discusses this as a campaign option — a monster can spend XP equal to the XP needed to advance one level to reduce its effective LA by 1 (minimum +0). This rewards adventuring and helps high-LA monsters stay relevant.
