# Saving Throws (3.5e)

Save = d20 + base save + ability mod + misc bonuses. DC typically 10 + spell level + caster's key ability mod.

## Base Save Progressions by Class

| Class | Good Saves | Poor Saves | Formula |
|---|---|---|---|
| Barbarian | Fort | Ref, Will | Good: 2 + ½ level; Poor: ⅓ level |
| Bard | Ref, Will | Fort | |
| Cleric | Fort, Will | Ref | |
| Druid | Fort, Will | Ref | |
| Fighter | Fort | Ref, Will | |
| Monk | Fort, Ref, Will (all good) | — | |
| Paladin | Fort | Ref, Will | +Cha to all saves |
| Ranger | Fort, Ref | Will | |
| Rogue | Ref | Fort, Will | |
| Sorcerer | Will | Fort, Ref | |
| Wizard | Will | Fort, Ref | |

**Good save formula:** +2 at level 1, then +½ level (level 1=2, 4=4, 8=6, 12=8, 16=10, 20=12)  
**Poor save formula:** ⅓ level rounded down (level 1=0, 3=1, 6=2, 9=3, 12=4, 15=5, 18=6, 20=6)

## What Each Save Protects Against

**Fortitude (Con-based):** Physical toughness. Poisons, diseases, death effects, energy drain, massive damage (Fort DC 15 or die when taking 50+ damage in one hit), paralysis from physical sources, petrification. Also: suffocation, starvation, forced march fatigue.

**Reflex (Dex-based):** Dodging area effects. Fireball, breath weapons, traps, pit falls, anything that can be dodged. Evasion (rogue/monk) negates on success; improved evasion negates on failure too.

**Will (Wis-based):** Mental fortitude. Mind-affecting spells (charms, compulsions, fear, phantasms), illusions (some), some death effects, domination, confusion, sleep. Paladins and clerics are typically strong here.

## Bonus Types That Stack

Saves follow normal stacking rules. **Resistance bonuses** (cloak, spell) stack with nothing of the same type. **Morale bonuses** (heroism) stack with nothing of the same type. **Deflection** is only for AC. Common stacking combos:
- Cloak of Resistance +2 (resistance) + Paladin Cha bonus (untyped) — stack
- Heroism (+2 morale) + Cloak of Resistance +2 (resistance) — stack
- Two cloaks of resistance — do NOT stack

## Saving Throw Riders for the GM

- **Concentration check** is not a save but uses a similar structure: d20 + Con + Concentration ranks vs DC 10 + damage taken + spell level.
- **Death saves** (massive damage): Fort DC 15 when taking 50+ damage in one hit, or instant death.
- **Energy drain:** Each negative level reduces saves by 1; Fort DC (10 + ½ HD + Cha mod) to remove after 24 hours.

*Sources: PHB 131-133, DMG 8.*
